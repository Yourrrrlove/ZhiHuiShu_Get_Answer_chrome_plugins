# Roadmap

## 2026 Q1

- Finish the local-first study library refactor
- Publish a compliant Chrome extension package focused on note-taking and review tracking
- Add a visible project policy that rejects answer-fetching, auto-fill, and exam-assistance features

## 2026 Q2

- Add JSON import and merge support
- Group entries by course and chapter
- Improve mobile-sized popup layout and accessibility labels

## 2026 Q3

- Add spaced-repetition review queues
- Add optional reminder prompts for items still marked `new` or `reviewing`
- Add export formats for Markdown and CSV

## Contribution rules

- Features must help learners organize and review their own understanding
- Features must not retrieve answers from third-party services
- Features must not automate quiz or exam workflows
- Any proposal that crosses those lines should be closed without implementation
