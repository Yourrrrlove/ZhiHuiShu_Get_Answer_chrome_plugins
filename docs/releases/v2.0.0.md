# v2.0.0

This release marks the transition from an old experimental codebase into a compliant study-helper project.

## Highlights

- Rebuilt the repository around a local-first Chrome extension
- Removed page injection, remote answer fetching, and exam-assistance logic
- Added structured study entries for questions, analysis, takeaways, tags, and review state
- Added search, status filters, deletion, and JSON export
- Rewrote the public README to document the project's educational scope and contribution boundaries
- Added a public roadmap for 2026 maintenance work

## Why this release matters

The project is now explicitly focused on helping learners capture their own explanations and review progress. It no longer interacts with course sites or third-party answer services.
